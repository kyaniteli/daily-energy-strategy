一、 下周调仓战术手册（2025.12.29 - 2026.1.2）
目标： 将神华的仓位平稳挪到长江电力，同时建立中广核和南瑞的底仓。
原则： 配对交易（Pair Trading）。不要卖了神华拿着现金等长电跌，卖出和买入要同步进行，防止踏空。
1. 长江电力 (600900) —— 核心替换
操作： 卖出中国神华的那一刻，立即（几秒钟内） 市价买入长江电力。
心态预警：
换仓后，如果神华大涨（因为股息高），长电不涨，千万别后悔。
你要告诉自己：“我卖出的是煤炭的周期风险，买入的是长江的永续水流。我支付了溢价，买到了安稳。”
2. 国电南瑞 (600406) —— 坚决建仓
价格锚点： 只要 PE 在 23倍 上下（股价约 23 元），不要犹豫。
操作： 直接买入 30% 仓位。这是你组合里唯一的进攻点，必须拿住筹码。
3. 中国广核 (003816) —— 无论价格
价格锚点： 3.85 元左右。
操作： 买入 30% 仓位。它是你的压舱石。
